Extremely fast disk usage analyzer.
Port of [ncdu](https://dev.yorhel.nl/ncdu) written in Go.

## Installation

Go:

    go get -u github.com/dundee/gdu


Arch Linux:

    yay -S gdu


## Usage

    gdu some_dir_to_analyze


## Running tests

    make test
